# User Manual

## Login
Use a role account (demo password `Password@123`): admin@, manager@, frontdesk@, housekeeping@, accounts@, corporate@hotel.com. The sidebar adapts to your role.

## Daily front-desk flow
1. **New enquiry** — Enquiries → *+ New Enquiry*; capture name, mobile, source, dates, budget, requirements, assign staff.
2. **Work the pipeline** — click any enquiry to open its detail: change status, read the history timeline.
3. **Schedule follow-ups** — Follow-Ups → *+ New*; filter *overdue* / *today* to clear your queue.
4. **Convert** — in the enquiry detail, pick an available room to convert to a booking (auto-creates guest + confirms enquiry).
5. **Check-in / Check-out** — Bookings → action buttons. Check-out auto-flags housekeeping and updates loyalty.

## Other modules
- **Availability** — date-range search shows only conflict-free rooms.
- **Housekeeping** — cleaning queue; mark rooms Dirty→Cleaning→Ready.
- **Room Service / Complaints / Feedback** — log and track to completion.
- **Corporate / Groups** — manage bulk accounts (coordinator role).
- **Guests / Loyalty** — stays, lifetime value, preferences, tier.
- **Shift Handover** — leave prioritized notes for the next shift.
- **Reports** — generate and export CSV (admin, manager, accounts).

## Dashboard
Ten live KPI cards, source/occupancy/revenue charts, conversion rate, and a recent-activity feed.
